**Cloudflare が好きな 26 歳。** デザイナー出身で、使いやすさを考えるのが好きです。

個人プロダクトを Cloudflare スタックで作っていて、いまは「AIChatClip」という Chrome / Firefox 拡張 + Web + API + Obsidian Plugin の SaaS を運用中。Cloudflare の新しい機能が出ると一通り試したくなる性格で、デザインはアートというより「こうしたら使いやすくなるよね」を考える方が好きです。

経営学部 → 起業準備 → デザイン → 個人開発、と歩いてきました。場面ごとに振っているように見えるかもしれませんが、自分のなかでは切り捨てて移った感覚はなくて、それぞれで身についたもの ── 経営学部で身についた事業視点、起業準備 2 年で身についた検証の習慣、デザインで身についた設計の理論 ── が、いまの個人開発のなかに道具として残っている感じです（デザインは引き続き勉強中です）。

AIChatClip では、開発に入る前に 1,000 円の課金意思を実際に確認してから着手しました。「作る前に売れるかを確かめる」みたいな感覚は、この辺の経緯から来ています。

## こんなことをしてきました

「これ作りたいな」を見つけて、技術で形にして、誰かに使われるまで、を規模を変えながら繰り返してきました。

- **AIChatClip** (個人プロダクト): AI チャットの会話を Obsidian / Notion / Webhook へ自動同期する SaaS。Cloudflare スタックで運用中。
- **金額入力の自動化** (業務改善): 週 3 時間 → 週 20 分。年間で 130 時間くらい減らしました。
- **カレンダー / LINE 自動化** (チーム支援): 同僚が予定登録に困っていたので、音声入力 × AI で自発的に組み立てたツール。
- **PL ダッシュボード** (既存ツール置き換え): Excel 製の見にくい PL シートを Cloudflare D1 + OOUI のダッシュボードに。Zero Trust でアクセス制御。
