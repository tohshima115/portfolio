## TL;DR

3 人チームの社会起業プロジェクト「Swept」に約 2 年関わって、ロゴ / ブランドアイデンティティ / 名刺 / Web の設計・実装と、ユーザーインタビュー / 仮説検証まで担当しました。自分にとっては「デザイナー」から「自分の設計を自分で実装する人」へ振り直した転換点になったプロジェクトです。方向性の違いで 2026 年 1 月に離脱しています。

| 期間 | 約 2 年 (2024 — 2026.01) |

## どんなチームだったか

3 人チームで「世の中に出すべきプロダクトって何だろう」を探りながら、デザインと企画と実装を一気通貫でやっていた期間です。最初はデザイナーとして入っていたんですけど、参画したタイミングがちょうど Cursor が出てきた時期で、「自分の設計を自分で実装した方が速いな」と思って担当範囲を広げていきました。当時の AI のクオリティはまだ荒削りだったんですが、「これは行ける」という手応えを早めに掴めたのは大きかったです。

プロジェクト自体は大きく 2 つのフェーズに分かれていました。

### Phase 1 — 旅行 (2024 — 2025 初頭・約 1 年)

最初はテーマを「旅行」に置いて、約 1 年かけてひたすら仮説検証をしていました。ユーザーインタビューを集中的にやって、リーンスタートアップ系の本を読みながら、現場で「沈黙の作り方」「定性と定量の切り分け」「カスタマージャーニーマップでペインを定義する」みたいな作法を覚えていった感じです。

最終的には、想定していた課題がターゲットにとって十分大きなペインに到達しない、というのが定量・定性の両面で見えてきたので、テーマを撤退・転換する判断をしました。**「ズルズル続けない」判断ができたこと** が、自分にとっては一番大きな学びかもしれません。

### Phase 2 — 愛着 (2025 春 — 2026.01)

テーマを「愛着 (attachment)」に切り替えて、このフェーズで自分は **ロゴ / ブランドアイデンティティ / 名刺 / Web の設計と実装** を中心に担当しました。

その後、プロジェクトが福祉・社会起業寄りの色合いを強めていく流れと、自分が向き合いたいスタートアップ像とのズレが大きくなってきたので、2026 年 1 月に方向性の違いを話し合った上で離脱しています。

## 担当したもの

### Logo / Brand Identity

"Swept" の名は「過去のトラウマや蟠りを掃き清める (Sweep away)」から来ています。

- **Sweep (掃く)**: 箒の動きや風の流れを抽象化した曲線で、滞った感情が流れ出す様子を表現
- **Connection (繋がり)**: 健全な他者とのつながりを、途切れない一筆書きのラインで象徴
- **Heal Green / Warm Gray**: 癒しと包容を表すカラーパレット

### 名刺 / DTP

ロゴから派生して、名刺の組版・入稿まで一人で対応しました。Web に閉じない仕事も一人で担保できる体制になっています。

### Web (Design + Implementation)

- 安心感のあるカラー設計と、読み手の負担を減らすタイポグラフィ
- 悩みの種類 / 支援者向け など属性別の導線設計
- **Astro** + Tailwind での実装

### UX Research

Phase 1 の検証期間でユーザーインタビューを何度も回しました。インタビュー設計、沈黙の使い方、カスタマージャーニーマップによるペイン定義あたりを、現場で動かしながら覚えていった感じです。

## 振り返り

- 「デザインまで」だった自分が「実装まで」になる転換点になった
- ロゴ → 名刺 → Web の一貫した世界観を作る経験ができた
- 仮説検証 (ユーザーインタビュー / ペイン定量化 / 撤退判断) の基本動作を、現場で実際に手を動かして覚えられた

> Phase 1 で身につけた仮説検証の作法は、その後の AIChatClip にもそのまま効いています。MVP 段階で 1,000 円の課金意思を先に確認してから実装に入れたのも、ここで手触りを覚えていたからだと思います。
