## 何を作っているか

Focus + Lock + Clock。SNS や動画サイトなど「時間泥棒」になりがちなサイトをブロックする拡張機能です。AIChatClip と同じく Cloudflare 上に一人で構築していて、拡張機能本体 (WXT + React) に加えて、マーケティングサイト・管理画面・API を Cloudflare Workers に同居させています。

```
apps/
  extension/   WXT + React (Chrome/Edge/Firefox)
  web/         React Router v7 on Workers Static Assets (foclock.app)
  admin/       React Router v7 on Workers Static Assets (admin.foclock.app)
  api/         Hono on Cloudflare Workers (api.foclock.app)
packages/
  shared/      共通の型定義・定数
  ui/          共通UIコンポーネント
```

認証は Better Auth (Google OAuth) で AIChatClip と同じ選定、課金は Stripe のライフタイム買い切り一本 (サブスクなし)。DB は Cloudflare D1 + Drizzle ORM です。

## 設計上の判断: 同期にサーバーを噛ませない

Pro 機能のクロスデバイス同期は、独自サーバーを経由せず `browser.storage.sync` (Chrome/Edge は Google、Firefox は Mozilla アカウントの同期領域) にそのまま乗せています。Foclock の API はこの同期に一切関与しません。**同期のためのサーバーとその運用を丸ごと持たずに済む** という判断です。
