## Web を Astro で作った話

実装は Astro + Tailwind。当時の選定理由を素直に書くと、

- Cloudflare に置きたかったので Next.js は相性的になし
- Vue 系を知らないので React 系で探す
- ただのコーポレートサイトに Next ほどの装備は要らない、極論 HTML/CSS で十分
- 「もっとシンプルでよくない?」という気持ちが当時の自分に強かった

で、Astro が一番きれいに収まりました。今もう一度同じ要件で選んでも、たぶん第 1 候補は Astro のままです。
