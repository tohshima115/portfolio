## Astro を選んだ理由

コーポレートサイトの実装は Astro + Tailwind。選定理由は、

- Cloudflare に置きたく、Next.js は当時相性が悪かった
- ただのコーポレートサイトに Next.js の装備は過剰。極論 HTML/CSS で足りる要件だった
- React 系の中で、最小構成で組めるものを探した

結果 Astro が一番きれいに収まりました。同じ要件なら今でも第 1 候補は Astro です。
