## テーマが「愛着」に決まるまで

旅行を畳んだあと、「じゃあ次どうする?」となったときに、自分から「愛着というテーマが人生のど真ん中にある」という話をして、CEO 役の人が「お前が納得するんやったらそれしかないんちゃう」と返してくれて決まったのが Phase 2 のテーマです。

ここから自分は **ロゴ / ブランドアイデンティティ / 名刺 / Web の設計と実装** をまるっと担当しました。

## プロジェクト名「Swept」について

ちなみに名前の由来は「掃き清める」みたいな話ではなくて、**Sweep のもう一つの意味「魅了する・席巻する」** から来ています。「ワクワクするものを作りたいよね」というところから付いた、というくらいの距離感で、そんなに深い意味は最初から込めていません。語感が良かった、というのも普通に大きいです。

## ロゴ: 卵 + S + 抱擁を黄金比の円で

ロゴ自体は **愛着** をど真ん中に置いて作りました。

モチーフは 3 つを重ねています:

- **卵**: 守るべきものであり、「コロンブスの卵」的な発想の源でもあり、「自分たちもまだ卵」これからどんどん変わっていく、という意味
- **S**: Swept の頭文字
- **抱擁**: 抱きしめる動き

形は **すべて黄金比で作った円の組み合わせだけで構成** していて、後から誰が見ても再現できるロゴになっています。

実は 1 年目のかなり初期にもロゴは一度作っていたんですが、SNS のアイコンに置いたときにグラデーションが強すぎたり書き込みすぎていたりで明らかに良くなかったので、Phase 2 でシンプルに作り直しました。正方形のアイコン枠にきれいに収まる、SNS 耐性のある形を意識した、というやつです。個人的にもかなり気に入っています。

ブランド設計の入り方としては、いつも **キーワード抽出 → マインドマップ** から始めるタイプです。これ無しでは進められない。Swept のときも「我々の今の状況」「扱うもの」と相性のいいモチーフを連想で広げていった結果、卵が出てきた、という順番でした。

## 名刺 / DTP

ロゴから派生して名刺の組版・入稿もやりました。Web 育ちなのでトンボとか CMYK は「あー、これか」と思いながらやった記憶はあるものの、特に詰まるところはなかったです。
