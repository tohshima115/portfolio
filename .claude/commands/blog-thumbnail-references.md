# サムネイル用 人物リファレンス集

`/blog-thumbnail` で人物を入れる際に使う、Freepikストック素材のリファレンス集。

## 使い方

1. コンセプトに合うポーズをこの表から選ぶ
2. `mcp__magnific__stock_download` に `id` と `type: vector` を渡してダウンロードURLを取得する
3. `mcp__magnific__creations_upload_image` にそのURLを渡してアップロードし、`identifier` を得る
4. `images_generate` の `references` に `{ "type": "style", "identifier": "<上記identifier>" }` を追加してプロンプトを投げる（**まだ実物ファイルはローカルに保存しない**。参照はFreepik側のIDだけで完結する）
5. 生成された人物がリファレンスの雰囲気に寄っているか確認する。寄っていなければ複数リファレンスを併用してみる（`references` は最大12件まで指定可能）

プロンプト本文には「following the reference pose/silhouette」のように、リファレンスのポーズ・シルエットに寄せる旨を一言添えること。色は必ずこのサイトのグリーン系パレットに変換する指示を出す（リファレンス画像は色付きだが、それは踏襲しない）。

## 注意点

- Freepikのプレミアム素材も `stock_download` で取得できることを確認済み（`248591276` など）
- リファレンスはあくまで**ポーズ・シルエットの参照**であり、服の色や髪型までは踏襲しなくてよい。むしろプロンプト側で「このサイトのグリーン系パレットのみに変換する」と明記しないと、リファレンスの色がそのまま出てしまうことがあるので注意
- 候補が足りない・合うポーズがない場合は `mcp__magnific__stock_search` で `isometric character <ポーズ名> flat illustration` のように検索して都度追加してよい。追加したら本ファイルの表にも一行足しておく
