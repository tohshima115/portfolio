# タスク: ブログサンプルの作成

## 概要
ユーザーからのリクエストにより、ブログのサンプル記事を3つ作成する。それに伴い、ブログ記事を表示するための設定（スキーマの定義、一覧ページの修正、詳細ページの作成）を行う。

## 目的
* プレースホルダー状態になっているブログ機能を利用可能な状態にする。
* サンプルデータを追加して、一覧表示と詳細表示が正しく動作することを確認する。

## 要件
* `src/content/config.ts` で `blog` コレクションのスキーマを定義する。
* `src/content/blog/` ディレクトリに3つのサンプル記事（Markdown）を作成する。
* `src/pages/blog/index.astro` でサンプルの記事がリスト表示されるように修正する。
* `src/pages/blog/[...slug].astro` を作成し、記事の詳細が表示されるようにする。
