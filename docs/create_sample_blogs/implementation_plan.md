# 実装計画

1. **コレクションスキーマの追加 (`src/content/config.ts`)**
   * `blog` コレクションの定義を追加する。
   * タイトル、公開日 (`pubDate`)、説明、オプションの画像を含むスキーマとする。

2. **サンプル記事の作成 (`src/content/blog/`)**
   * 3つの Markdown 形式の記事を作成する。
   * それぞれに適切なフロントマター（タイトル、日付など）と本文（ダミーコンテンツ）を記載する。

3. **ブログ一覧ページの修正 (`src/pages/blog/index.astro`)**
   * プレースホルダーの `posts` をコメントアウトし、実際に Astro の `getCollection('blog')` から記事を取得して表示する。
   * 記事リストの UI コンポーネントを埋め込む。

4. **ブログ詳細ページの作成 (`src/pages/blog/[...slug].astro`)**
   * 個々のブログ記事を表示するためのコンポーネントを作成する。
   * `getStaticPaths` 関数でコレクション内容からパスを生成。
   * 記事のタイトル、日付、本文のレンダリングを行う。
