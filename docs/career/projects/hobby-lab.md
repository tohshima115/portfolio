# 実験と遊び (Hobby Lab) — 質問リスト (ノリ更新版)

> サイトソース: `src/content/projects/hobby.mdx`
> 音声メモ: `docs/career/projects/voice-memos/hobby-lab-*.md`

## このリストの方針

今の mdx は「復活の呪文ジェネレーター」1 個だけ載ってて、それ以外は「Hobby はこういう場です」の説明文。Swept の mdx を書き直したのと同じノリで、本人が遊んでる感とエピソードを引き出したい。

- 「何を作ったか」より「なんで作ったか」「作ってる時何が楽しかったか」が欲しい
- 真面目に書こうとせず、ニヤニヤ書ける温度感を維持
- 出せる作品は出す、出さない (=出してないけど存在する) ものは存在だけ匂わせるのもOK
- AI ツールの素振り / Cloudflare 新サービス検証みたいな「ちゃんと役に立ってる遊び」も Hobby 扱いでOK
- 一気に答えなくていい。気が向いたものから

各質問に `[ちゃんと]` / `[ゆるくでOK]` を付けてあります。
