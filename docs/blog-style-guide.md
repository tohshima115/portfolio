# ブログ スタイルガイド

このサイトのブログに投稿する際の文体・トーンのガイドラインです。

## 文体の基本

**です・ます調を使う。**  
である調は堅苦しく、自分が読んでいられない。です・ます調の方が読みやすいし、話しかけるような距離感に合っている。

## トーンの方針

**等身大で書く。**  
就活っぽい・自己PR的な文章にしない。「私はこれだけできます」という売り込みトーンは鼻につくので避ける。読んで「なんかこいつ、自分のこと書いてるんだな」と思える感じを目指す。

**飾らない。**  
整えすぎない。語り口に近い自然さを残す。「まあ、そういう話でした」みたいな締め方でいい。かっちりした結論を無理に作らない。

**自分の感覚を主語にする。**  
「一般的に〜」より「自分は〜が嫌だった」「〜が好きだった」の方が読める。

## やらないこと

- 「貴重な経験をしました」「大変勉強になりました」みたいな研修報告文体
- 「〜と言えるでしょう」「〜が重要です」みたいな評論・解説文体
- 「本記事では〜について解説します」みたいな技術ブログの導入テンプレ
- 箇条書きで全部まとめようとする（会話の流れで書いた方が読める）

## 見出しの使い方

- `##` を区切りに使う。1記事に3〜5個くらいが目安。
- 見出しは「問いの形」か「内容の要約」にする。
  - 例：「なぜ作ったか」「一番困ったこと：〜」
- 「まとめ」「おわりに」で締めなくていい。最後の段落が締めになっていればそれでOK。

## 長さの目安

- 1記事：800〜2000字くらい
- 長くなりそうなら記事を分ける
- 短くても気にしない。伝えたいことが伝わればいい

## AIで下書きを作るとき

AIが生成したテキストは就活・自己PR調になりがち。必ず読み直して、自分の語り口に直す。  
「自分がこのサイトを見ていられるか」が基準。
