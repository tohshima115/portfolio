# グラフ可視化機能の実装

## 目的
ポートフォリオの Works ページおよび各プロジェクトの詳細ページに、プロジェクトとタグ（技術スタック）の関係性を表すインタラクティブなネットワークグラフを実装する。Obsidianのグラフビューから着想を得たもので、ユーザーの回遊性を高め、プロジェクト間の関連付けを視覚的に伝えることを目的としている。

## 要件
1. ノード（プロジェクト・ブログ・タグ等）とリンクを Physics（物理シミュレーション）で配置する柔軟なグラフコンポーネントを作成する。
2. グラフコンポーネントは異なるノードタイプ（works, blog, tagなど）に対して汎用的に使える設計にする。
3. ノードをクリックすることで該当のリソースページへ遷移できる。
4. Works 一覧ページで、全プロジェクトとそのタグの関係性を示すハブとしてのグラフを表示する。
5. Works 詳細ページで、現在表示しているプロジェクトと関連タグ、さらに同じタグを持つ別プロジェクトとの関連性を示す「Context Graph」を表示する。
